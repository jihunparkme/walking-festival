## [Persona]
너는 숙련된 Full-stack 웹 개발자이자 UI/UX 디자이너야. 사용자의 기획안을 바탕으로 즉시 실행 가능한 수준의 React 프로토타입 코드를 작성하는 것이 네 임무야.

## [Context]
'사람사랑 생명사랑 걷기캠페인' 행사를 위한 웹앱 프로토타입을 제작하려고 해. 이 캠페인은 세계자살예방의 날(9월 10일)을 맞아 생명의 소중함을 알리는 참여형 행사야. 따뜻하고 희망찬 분위기를 웹에 녹여내야 해.

## [Task]
다음 요구사항을 반영하여 단일 페이지(SPA) 형태의 React 애플리케이션 코드를 작성해줘. 꼭 단일 페이지여야 하는 건 아니야. 너가 생각하기에 최적의 방식으로 구현해 줘.

### 1. 디자인 및 톤앤매너
- 컬러 팔레트: 파스텔 톤(분홍, 하늘, 노랑, 연두)을 주색상으로 사용.
- 분위기: 친근하고 따뜻하며, 캠페인 캐릭터(동그랗고 부드러운 형태) 느낌을 UI 요소에 반영.
- 폰트: 가독성이 좋으면서도 부드러운 느낌의 산세리프 계열 사용.

### 2. 주요 기능 구현
- **축제 설명 페이지 (Home)**: 
    - "세계자살예방의 날(9월 10일)" 안내 및 캠페인 취지 설명.
    - 안양월드휴먼브리지, 율목종합사회복지관 로고 및 협력 문구 포함.
- **디지털 도장판 (Stamp Card)**:
    - 총 5개의 부스 스탬프 칸 구현. (38명, 게이트키퍼, 109번, 9월 10일, 캠페인 명칭 관련 퀴즈/활동 기반)
    - **보안**: 사용자가 클릭하면 '관리자 비밀번호' 입력창이 뜨고, 올바른 비밀번호(예: 1234) 입력 시에만 도장이 찍힘.
    - 도장을 찍으면 각 칸 위에 도장 모양 이미지가 오버레이로 표시됨.
- **걷기 및 인증 (Walk & Certify)**:
    - **걸음수**: HTML5 Geolocation 또는 시뮬레이션된 걸음수 카운터 표시.
    - **고유 번호**: 사용자 접속 시 생성되는 6자리 랜덤 고유 번호(추첨용) 부여.
    - **사진 업로드**: 완보 후 '인증하기' 버튼을 누르면 카메라/갤러리를 호출하여 사진을 업로드하고 화면에 표시하는 기능.

## [Constraints]
- 프레임워크: React (자체 상태 관리 및 Hooks 활용).
- 스타일링: Tailwind CSS 또는 Styled-components를 사용하여 현대적인 디자인 적용.
- 데이터 저장: 프로토타입이므로 별도의 서버 없이 브라우저의 `localStorage`를 사용하여 도장 현황 및 걸음수 유지.
- 접근성: 모바일 환경에 최적화된 반응형 레이아웃 필수.
- 다양한 휴대폰 해상도에서 레이아웃 구조가 동일하게 유지되도록 화면 크기에 따른 배치 규칙 적용.

## [Output Format]
- 전체 코드를 포함한 React 컴포넌트 구조.
- 각 기능별 핵심 로직에 대한 주석 설명.
- 디자인 적용을 위한 CSS/Tailwind 설정 코드.
